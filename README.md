Este repositorio contiene el desarrollo de la práctica de campo número 3.
El objetivo es aplicar los conceptos de sobrecarga de métodos, manejo de errores y uso de colecciones en Java.
Cada ejercicio fue elaborado de forma individual y cuenta con comentarios explicativos dentro del código.

Archivos incluidos en el repositorio:

CalculadoraPregunta1.java
Contiene la clase Calculadora que demuestra el uso de métodos sobrecargados para sumar distintos tipos y cantidades de números.

DivisionSeguraPregunta2.java
Programa que realiza una división entre dos enteros ingresados por el usuario, controlando los errores por división entre cero y entradas no válidas.

ListaEstudiantesPregunta3.java
Ejemplo de uso de ArrayList para registrar, eliminar y mostrar una lista de nombres de estudiantes.

InventarioPregunta4.java
Implementación de una clase Inventario que usa sobrecarga de métodos para agregar productos con distintos parámetros y validaciones para evitar valores negativos.

NumeroNegativoPregunta5.java
Programa que solicita un número entero y maneja dos tipos de excepciones: entrada no numérica y número negativo mediante una excepción personalizada.

Para ejecutar los programas, basta con compilar y correr cada archivo de forma independiente en la terminal o desde un entorno como Eclipse o Visual Studio Code.

javac NombreDelArchivo.java
java NombreDelArchivo

Ejemplo:
javac CalculadoraPregunta1.java
java CalculadoraPregunta1

Este trabajo fue realizado con fines académicos para reforzar el uso de la programación orientada a objetos y el control de excepciones en Java.
